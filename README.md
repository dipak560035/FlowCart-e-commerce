# FlowCart - Premium E-commerce Platform

A modern, fully-featured e-commerce platform built with Next.js 15, TypeScript, Tailwind CSS, GSAP, TanStack Query, and Redux Toolkit.

## Live Demo

[View Live Demo](https://flow-cart-e-commerce.vercel.app/)

## Tech Stack

| Category | Technology |
|----------|------------|
| Framework | Next.js 15 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS + Custom CSS |
| Animations | GSAP (GreenSock) |
| Data Fetching | TanStack Query |
| State Management | Redux Toolkit |
| Icons | Lucide React |
| Charts | Recharts |

## Setup Instructions

### Prerequisites

- Node.js 18.x
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/dipak560035/FlowCart-e-commerce
# Navigate to project directory
cd flowcart

# Install dependencies
npm install

# Run development server
npm run dev
